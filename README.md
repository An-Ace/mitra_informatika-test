
# Dokumentasi Test PT. Mitra Informatika



## Install

Untuk install Apps:

```bash
  npm install
```

## Run Local Server Apps

Untuk install Apps:

```bash
  npm start
```
##### OR
```bash
  npm run dev
```

#### Untuk membuka Web Buka pada URL berikut:
